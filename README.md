About
-------
This plugin is for LabAPI version 1.1.4.1 / Exiled Version 9.10.1

This adds the ability to customize your own name and character info to whatever name you so desire in SCP SL.

Commands
--------
.n /.name < name > | Max 32 characters

.info / .cinfo < imnfo > | No Limit on characters

Installaton
------------
1. Extract File

2. Go to 'Bin/Debug/Net48/JFDCustomNameInfo.dll'

4. Paste DLL File into Exiled/Plugins

5. Restart Server

6. Have Fun!!
